# Simple Shell

This is a simple Unix shell written in C that allows users to execute basic commands. It's designed to be a lightweight and easy-to-use alternative to the standard Unix shell.

## Installation

To install and use the simple shell, follow these steps:

1. Clone the repository: `git clone https://github.com/krmohammed/simple_shell.git`
2. Compile the code: `gcc -Wall -Werror -Wextra -pedantic *.c -o hsh`
3. Run the shell: `./hsh`

The simple shell supports a variety of basic commands, including:

- `cd`: Change directory
- `ls`: List files in a directory
- `echo`: Print a message to the screen
- `exit`: Exit the shell

You can also execute external programs by typing the name of the program followed by any necessary arguments.

## Built With

This project was built using the C programming language and the GCC compiler.
